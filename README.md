## 🚙 수현이랑
**완벽하게 엄빠 몰래 가는 여행 서비스**
```
여행을 위해 같이 사진을 찍을 수현이 찾기, 여행 관련 사진을 공유하는 갤러리, 수현이 찾기 후 수현이를 만나기 위한 채팅 기능 등을 제공하며,

부모님과 자녀 간의 이성교제 관련 소통이 자유롭게 이루어지는 날을 한층 가깝게 만드는 것을 목표로 합니다 🙂
```
